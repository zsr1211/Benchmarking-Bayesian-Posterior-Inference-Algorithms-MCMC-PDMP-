
# Exploratory analysis runs

The additional posterior sampling runs used or considered for the exploratory random forest analysis are hosted on OSF because several individual result files exceed GitHub’s file-size limit.

All posteriors included in the exploratory random forest analysis have reference draws available, either from PosteriorDB or from reference sampling runs generated for this study.

The exploratory random forest analysis combines selected runs from two sources:

1. a subset of the main benchmark runs available in `main analysis runs/`; and

2. a subset of the additional posterior runs stored on OSF.

Not all runs stored on OSF were included in the final exploratory analysis. The analysis script applies additional selection criteria, including the availability of suitable reference information and successfully completed algorithm runs.
